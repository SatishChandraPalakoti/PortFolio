EASY COOK
About :
Easy cook is a culinary platform. An IBM Bluemix hosted web-application that integrates the features from multiple web sites. Easy cook interacts with APIs of BigOven, YouTube and Spoonacular which aided to acquire Recipe and directions, Videos to help cooking and Graphical representation of the calorific values of the ingredients. These REST API's provide the information that the user requests from the search. The security for Easy Cook has been taken a help from Google OAuth2.0.
